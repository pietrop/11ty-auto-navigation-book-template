# IIFE

An IIFE (Immediately Invoked Function Expression) is a JavaScript function that runs as soon as it is defined.

```jsx

```

```jsx
(function foo() {
  console.log('my log message');
})();
// or function can be anonymous
(function() {
    console.log('my log message');
})();
```

Passing arguments to `IIFE`

```jsx
(function(a, b) {
  // a == 'Hello'
  // b == 'IIFE'
})('Hello', 'IIFE');
```

`IIFE` "interface"

```jsx
const test = (function (){
	return {
		hello: function(){
				console.log('world')
		}
	}
})();

test.hello()
```
